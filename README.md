# 🌐 Mapa Interactivo de Correlativas — Tecnicatura en Desarrollo Web

Aplicación web interactiva para el seguimiento del plan de estudios de la **Tecnicatura Universitaria en Desarrollo Web** de la UNdeC (Universidad Nacional de Chilecito).

🔗 **[Ver aplicación en vivo](https://zzznata.github.io/Mapa-Interactivo-de-Correlativas-Tecnicatura-en-Desarrollo-Web/)**

---

## ✨ Funcionalidades

- 📍 **Mapa visual** de las 16 materias organizadas por año y cuatrimestre
- 🔗 **Correlativas interactivas** — al seleccionar una materia se destacan sus requisitos y las materias que desbloquea
- 👤 **Sistema de usuarios** — registro, login y cambio entre perfiles
- 📊 **Seguimiento de progreso** — marcá cada materia con su estado actual:
  - ⬜ Sin cursar
  - 🔵 Cursando
  - 🟢 Aprobada
  - 🔴 Reprobada
- ✅ **Validación de correlativas con doble distinción** según Resolución Rectoral N° 356-25:
  - **Para cursar** → las correlativas requeridas deben estar regularizadas (cursando o aprobadas)
  - **Para rendir** → las correlativas requeridas deben estar aprobadas
- 💾 **Persistencia local** — el progreso se guarda automáticamente en el navegador
- 📈 **Contador de progreso** — materias aprobadas en tiempo real

---

## 🚀 Cómo usar

1. Abrí la aplicación en el navegador
2. Registrate con tu nombre y apellido
3. Hacé click en cualquier materia para ver sus correlativas y cambiar su estado
4. Tu progreso se guarda automáticamente — podés cerrar y volver cuando quieras

## 🛠️ Tecnologías

- **HTML5**

- **localStorage** para persistencia de datos
